## About the Application Tech Stack

- This project is an **AI-powered ecommerce gaming application** built with **React Native** (frontend) and **Django with Python** (backend).
- An integrated AI recommendation system that analyzes the user behavior, preferences, and interaction history to deliver **personalized game and product suggestions**.
- The React Native frontend includes optimized UI/UX for web browser, reusable components, and responsive layouts tailored for a gaming-focused experience.
- The Django backend handles Database connections paired with Mongo DB, will be applying secure authentication, some product management, in the near future will be adding the teafutre of order processing.
- Communication between the frontend and backend follows a **RESTful API** pattern for consistent and scalable data exchange.
- The application is designed with a **modular architecture**, making it easy to extend with new AI models, product categories, or gaming features.
